Neural Network for Character Recognition

The very first neural net that I wrote was a MATLAB program that took an MNIST dataset to train on and outputted the character that most closely matched an input image. I wrote this barebones net by coding the regularized logistic regression cost function and the gradient function for backpropagation. The accuracy of this net turned out to be surprisingly high for a net with only 1 relatively small hidden layer of 25 nodes (~95% accuracy).
